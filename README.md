## 📲 Установка

> **Требования:** Android 8.0+ (API 26)

```
1. Скачайте APK из Releases
2. Установите на телефон
3. Настройки → Язык и ввод → Включите TAJKEYBOARD AI
4. Выберите как клавиатуру по умолчанию
5. Готово!
```

<br>

---

## 🔑 Активация ИИ

Клавиатура работает без ключа: раскладки, T9 офлайн, эмодзи, буфер.

Для ИИ-функций (исправление, перефраз, перевод) нужен ключ:

> 1. Откройте **[@TAJGROUP_License_bot](https://t.me/TAJGROUP_License_bot)**
> 2. Нажмите «Получить ключ» → «Клавиатура AI»
> 3. Подпишитесь на канал и группу
> 4. Скопируйте ключ `XXXX-XXXX-XXXX-XXXX`
> 5. Вставьте в приложении: **Настройки → Личный кабинет**
