## 📲 Установка

```bash
# 1. Скачайте APK из Releases (кнопка выше)
# 2. Установите на Android 8.0+
# 3. Включите клавиатуру:
Настройки → Язык и ввод → TAJKEYBOARD AI → Включить
# 4. Выберите по умолчанию
# 5. Готово! ✓
```

<br>

---

## 🔑 Активация ИИ-функций

> Без ключа работает: раскладки, T9 офлайн, эмодзи, буфер.
> С ключом: ИИ-исправление, перефразирование, перевод.

```
1 → Откройте @TAJGROUP_License_bot в Telegram
2 → Нажмите «Получить ключ» → «Клавиатура AI»  
3 → Подпишитесь на канал и группу
4 → Получите ключ: XXXX-XXXX-XXXX-XXXX
5 → Настройки → Личный кабинет → Вставьте ключ
```
